## Cinemaddict - online movie app

**Technologies used:** JavaScript (ES6), Webpack, Day.js, Apexcharts.js.

Install deps

```sh
npm i
```

Run dev mode

```sh
npm run start
```

Run prod mode

```sh
npm run prod
```
