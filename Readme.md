## Проект онлайн кинотеатра Cinemaddict

**Technologies used:** JavaScript (ES6), Webpack, Day.js, Apexcharts.js.

Установка зависимостей

```sh
npm i
```

Запуск в dev режиме

```sh
npm run start
```

Запуск в prod режиме
```sh
npm run prod
```
